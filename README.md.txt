# 🌦️ Weather Data Analysis using NumPy

This project simulates 30 days of weather data (temperature, humidity, rainfall) 
using NumPy and performs basic statistical analysis.

## 🔍 What’s Inside
- Simulated weather data with NumPy
- Stats: Mean, Max, Std Dev, Rainfall Count
- Humidity Comfort Index
- Easy, beginner-friendly analysis

## 📚 Tech Used
- Python 3
- NumPy
